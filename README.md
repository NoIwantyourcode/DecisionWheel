# Welcome to Decision Wheel Maker!
Have you ever been faced with a complex problem where all the solutions are equally good or equally bad and you are unable to decide what to choose?
Well, now with this decision wheel spinner you can add your own options and click the wheel to spin so it can choose for you!

# Features of this Decision Wheel
This decision wheel features a history where you can see the results of your past spins! It will also store your last used choices so that if you need, you can use them next time with ease! This decision wheel is completely fair, In no way will it be biased to particular answer. There is also a theme changer at the bottom of the page where you can select your own themes and color for the website!

# Tech stack
This project features 3 files, all with different roles, The app.js file is used to run the logic and make the webisite function, it makes the wheel spin, buttons work and somewhat helps in styling the website. The style.css file is used to style the website to the I want it. Over here you will find all the styling information I used to style the website. The main.html file combines the app.js file and style.css file to make the website work, It also adds structure and layout necessary for the website.

# How to contribute
If you want to contribute to this project,

1. Clone/fork this project to get a copy of the code
2. make the changes you want to it based on the tech stack I have written above (app.js = logic, style.css = styling and themes of the website, main.html = structure and layout of the website)
3. Once you have made your changes, tested to see if they work. you can go ahead and submit a pull request to merge those changes here.
